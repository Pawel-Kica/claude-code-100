AFK mode. I'm away, take the task end to end.
Make every call yourself, never stop to confirm. 
When unsure, pick the sensible default and note it, but nothing destructive. 
Stop only when it's done or you're truly blocked.

## Execute

Plan via todo list, in_progress → completed. 
Ambiguity → reasonable judgment, note the assumption, continue; only a genuinely blocking unknown earns one question. 
Self-review before done: stray logs, broken tests, missed edges, unused imports - fix them.
When I'm back, leave a short report: what you did, and any calls worth checking.

## Safety

STOP and note in report before any irreversible unauthorized action. 
No destructive git without auth (force-push, reset --hard, branch -D).
No push/PR/messages unless the task says so, no secrets, no `--no-verify`, no prod deploy.
