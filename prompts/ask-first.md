# ask-first

Stops the agent from guessing. Makes it show a plan before it touches anything.

```
/grill-me if you have any questions, but don't ask stupid questions.

After resolving all questions, show me the steps you're about to take. Wait for my OK, then go.
```

When you don't know where you'll land, swap the first line for:

```
/grill-me relentlessly. I'm not sure where we'll land, so ask many questions.
```

Needs the `grill-me` skill installed.
